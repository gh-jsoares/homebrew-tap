# Homebrew Tap

Custom [Homebrew](https://brew.sh) tap for gh-jsoares projects.

## Install

```bash
brew install gh-jsoares/tap/dotctl
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [dotctl](https://github.com/gh-jsoares/dotctl) | Developer environment orchestrator for macOS |

## Updating

Formulae are automatically updated on each release. Run:

```bash
brew upgrade dotctl
```
